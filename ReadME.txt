1. unzip these files to a folder.
2. open terminal/command prompt
3. run this command - "yarn install" 
4. after installation done then run this command - "yarn dev"

FYI - 
* If node is not installed on your device then please download and install node from this link. https://nodejs.org/en/download/current 
* If yarn command is not found then run this command - 'npm install --global yarn'
